## 第一章：快速判别——症状分类

| 症状表现 | 最可能原因 | 紧急程度 |
|---------|-----------|---------|
| **按电源键完全无反应** | 电源适配器/电池/主板供电回路 | ★★★★★ |
| **电源灯闪一下就灭** | 电池耗尽/主板短路/电源键故障 | ★★★★★ |
| **开机几秒（<10秒）就断电** | 电源键短路/内部短路/主板故障 | ★★★★☆ |
| **开机十几秒到1分钟关机** | 内存故障/BIOS损坏/供电不足 | ★★★★☆ |
| **开机1-5分钟关机** | CPU过热/散热故障/电源不稳 | ★★★☆☆ |
| **能进BIOS但进不了系统** | 硬盘故障/系统损坏/驱动问题 | ★★★☆☆ |
| **进系统后几分钟关机** | 驱动冲突/病毒/电源管理设置 | ★★☆☆☆ |
| **不定时随机关机** | 电源电压不稳/电容老化/接触不良 | ★★★☆☆ |

## 第二章：零成本排查（无需拆机）

### Step 1：释放静电（成功率极高，机械革命常见解法）

**原理**：笔记本内部积累静电可导致各种异常行为，包括无法开机、开机即关。

**操作方法**：
1. 关机（如果还能关的话，否则长按电源键10秒强制关机）
2. **拔掉电源适配器**
3. **拔掉所有外接设备**（U盘、鼠标、键盘、Type-C线、HDMI线等全部拔掉）
4. **长按电源键20-30秒**（这一步是关键，目的是释放主板残余电荷）
5. 松开电源键，等待10秒
6. 插上电源适配器（先不插其他设备）
7. 按电源键正常开机

> ⚡ **机械革命CODE01用户实测**：释放静电后直接恢复正常，这是最简单有效的第一步！
> ⚡ **注意**：如果用电环境是老房子、自建房、没有接地线，静电问题会更频繁。

### Step 2：检查电源适配器

1. 检查适配器指示灯是否亮（如果有的话）
2. 检查电源线是否有破损/折痕
3. 检查DC插口/Type-C充电口是否松动、有异物
4. 如有条件，**换一个同规格电源适配器试试**
5. 尝试**拔掉电池，仅用电源适配器供电开机**（如果电池可拆）

> 🔋 有用户发现：电池故障导致按开机键只闪一下就灭，插上电源线后正常开机。

### Step 3：断开所有外接设备

外接设备可能导致功耗过高或兼容性冲突：
1. 拔掉所有USB设备
2. 拔掉外接显示器线
3. 拔掉网线
4. 拔掉SD卡
5. 仅保留电源适配器，尝试开机

### Step 4：检查开机键

**原理**：开关按钮内部短路是「开机几秒就关机」的经典原因之一。按钮一次短路=开机指令，再次短路=关机指令，如果按钮持续短路，就会出现开了又关的现象。

**测试方法**：
- 按开机键时注意手感，是否有粘滞/不回弹的情况
- 尝试快速轻按（不要长按）
- 如果按钮明显卡住，用压缩空气对准按钮周围吹

### Step 5：环境因素检查

1. **温度**：笔记本是否在高温环境中（>35°C）？是否放在被子/枕头等不透气的表面上？
2. **湿度**：是否潮湿环境？是否有水/液体溅入过？
3. **电源环境**：插座是否稳定？是否接了过多设备？是否使用排插？
4. **物理损伤**：笔记本近期是否跌落、被压、遭受撞击？

## 第三章：轻度拆机排查

> ⚠️ 拆机前：关机、拔电源、释放静电（触摸金属物体）

### Step 6：重新安装内存条（成功率高，占笔记本重启故障的40%以上）

**原理**：内存条松动或金手指氧化是导致开机异常的最常见硬件原因之一。

**操作方法**：
1. 拆开笔记本后盖（通常只需拧开几颗螺丝）
2. 找到内存条插槽
3. 扣开两侧卡扣，取出内存条
4. 用**橡皮擦**轻轻擦拭金手指（金色触点部分），清除氧化层
5. 确认插槽内无灰尘
6. 重新插入内存条，确保两侧卡扣到位（听到"咔嗒"声）
7. 如果有两条内存，**单条交替测试**（排除某一条故障）

> 📌 单条测试法：只插一条内存试能否正常开机，再换另一条试。如果一条正常一条不行，说明内存条坏了。
> 📌 换插槽测试：把内存条插到另一个插槽，排除插槽故障。

### Step 7：检查硬盘/SSD连接

1. 检查M.2 SSD是否松动
2. 拧紧固定螺丝
3. 如有条件，**拔掉硬盘后尝试开机进BIOS**（如果能进说明硬盘可能有问题）

### Step 8：检查电池连接

1. 找到电池排线接口
2. 拔掉电池排线
3. 仅用电源适配器开机测试
4. 如果仅用适配器能正常开机 → 电池故障

> 🔋 机械革命Z3 Pro用户反映：电池3年损耗率24.4%，且电池偷电现象常见。

### Step 9：目视检查主板

用手电/手机闪光灯照看主板：
1. 是否有**烧焦/发黑痕迹**
2. 是否有**鼓包的电容**（电解电容顶部应平整，如果鼓起来说明坏了）
3. 是否有**松动的螺丝**掉在主板上（会导致短路！）
4. 是否有**液体残留/腐蚀痕迹**

> 💥 机械革命Z3 Pro真实案例：用户玩游戏时突然"啪一声冒烟"，一个电容烧了，需换主板。

## 第四章：深度硬件排查

### Step 10：清灰 + 更换硅脂

**原因**：灰尘堵塞散热鳍片 + CPU/GPU硅脂干裂 → 热保护关机

**操作方法**：
1. 拆下散热模组（风扇+散热铜管）
2. 用毛刷清理风扇扇叶和散热鳍片
3. 用无水酒精棉清除CPU/GPU表面旧硅脂
4. 涂抹新的高导热硅脂（推荐：信越7921、利民TF7等）
5. 重新安装散热模组，确保螺丝均匀拧紧

> 🌡️ 实测案例：机械革命笔记本清灰后温度从90°C+降到70°C以下，自动关机问题消失。

### Step 11：CMOS电池检测与更换

**症状线索**：关机后时间不保存、BIOS设置会重置、开机异常

**操作方法**：
1. 找到主板上的纽扣电池（CR2032）
2. 用万用表测量电压（正常应≥2.7V，低于2V建议更换）
3. 更换新电池
4. 开机后重新设置BIOS时间和参数

> ⏰ 机械革命Z3 Pro案例：CMOS电池只剩2.1V（正常3V），更换后时钟正常。

### Step 12：检查内部短路

1. 轻轻晃动笔记本，听是否有松动零件的声音
2. 检查所有排线是否正确连接（屏幕排线、键盘排线、电池排线等）
3. 检查是否有螺丝掉落在主板上
4. 检查散热铜管是否与主板某个元件异常接触

### Step 13：电源回路检测（需万用表）

1. 测量DC输入口电压是否与适配器标称一致
2. 测量电池接口电压
3. 测量主板关键供电点（需参考主板图纸）

## 第五章：Software/BIOS层排查

> 前提：能开机进入BIOS或能短暂进入系统

### Step 14：进入BIOS检查

**机械革命进BIOS方法**：开机时按 **F2**

检查项：
1. CPU温度是否正常显示
2. 风扇转速是否正常
3. 内存是否正确识别（容量、频率）
4. 硬盘是否正确识别
5. 启动顺序是否正确
6. 是否有异常的唤醒设置（USB Wake / LAN Wake → 建议关闭）

### Step 15：BIOS恢复默认设置

1. 进入BIOS
2. 找到 **Load Setup Defaults** 或 **Restore Defaults**
3. 保存并退出
4. 重启观察

### Step 16：进入安全模式测试

1. 强制关机3次触发Windows修复模式
2. 选择 **疑难解答 → 高级选项 → 启动设置 → 安全模式**
3. 如果安全模式下稳定不关机 → 问题出在驱动或软件

### Step 17：系统修复

如果能进入Windows修复环境：
1. **sfc /scannow** — 扫描修复系统文件
2. **chkdsk /f** — 修复磁盘错误
3. **DISM /Online /Cleanup-Image /RestoreHealth** — 修复系统映像
4. **启动修复** — 自动修复启动问题

### Step 18：检查系统日志

Win+R → eventvwr → Windows日志 → 系统
查找关机前的错误/警告事件，记录事件ID和来源。

### Step 19：关闭自动重启

右键此电脑 → 属性 → 高级系统设置 → 启动和故障恢复 → 设置 →
取消勾选「自动重新启动」→ 这样蓝屏时会停住显示错误代码而不是直接重启。

### Step 20：检查电源管理设置

控制面板 → 电源选项 → 选择「高性能」模式
更改计划设置 → 更改高级电源设置 → 检查所有休眠/睡眠/关机相关设置。

### Step 21：更新/回滚驱动

特别注意：
- **显卡驱动**（尤其是集显+独显切换导致的问题）
- **芯片组驱动**
- **电源管理驱动**
- **BIOS/EC固件**

> ⚠️ 机械革命电竞控制台(GCUService)已知Bug：待机时不断停止工作，新版已修复。

### Step 22：杀毒扫描

某些病毒（如冲击波病毒）会导致系统60秒后自动重启。
建议用U盘启动PE环境进行离线杀毒。

## 第六章：机械革命品牌特有问题

### 6.1 南桥芯片故障（Z2等早期型号）
- 南桥芯片(SR40B)裸露无辅助散热，长期高温易损坏
- 症状：各种随机故障、USB失灵、开机异常
- 解法：需专业维修更换芯片

### 6.2 电容烧毁（Z3 Pro等）
- 换大功率电源后持续高负载，电容可能爆裂
- 症状：啪一声冒烟，之后无法开机
- 解法：更换主板

### 6.3 USB/LAN唤醒导致关机后自动重启（CODE01等）
- BIOS中USB Wake和LAN Wake默认开启
- 症状：关机后自动重启，电源灯一直亮
- 解法：进BIOS → 将所有WAKE选项设为Disabled → F4保存

### 6.4 电池偷电（Z3 Pro等）
- 高负载时BIOS会从电池补充功率
- 修改EC配置可解决，但有风险

### 6.5 键盘/小键盘触发关机
- 按小键盘数字键导致关机（疑似静电问题）
- 解法：设置关机键为休眠，或检查键盘排线

### 6.6 CMOS电池寿命短
- 使用2-3年后CMOS电池可能不足
- 关机后时钟停止
- 解法：更换CR2032电池

### 6.7 Modern Standby (S0) 问题
- Windows现代待机模式导致异常唤醒和功耗问题
- 解法：BIOS中将S0改为S3（Suspend to RAM）

### 6.8 散热设计问题
- 散热垫可能无法完全覆盖内存等组件
- 游戏时内存过热导致死机
- 解法：加厚散热垫或添加散热贴

## 第七章：终极排查流程图

```
开机自动关机
│
├─ 按电源键完全无反应？
│  ├─ YES → 检查电源适配器 → 换适配器试 → 拆电池用适配器直供
│  │         → 如仍无反应 → 主板供电回路故障，需专业维修
│  └─ NO ↓
│
├─ 电源灯闪一下就灭？
│  ├─ YES → 释放静电(Step 1) → 换电源适配器 → 拆电池试
│  │         → 检查主板是否有短路(掉落的螺丝等)
│  └─ NO ↓
│
├─ 开机几秒就关(<10秒)？
│  ├─ YES → 释放静电 → 检查开机键是否短路(Step 4)
│  │         → 拔掉所有外设 → 重插内存条(Step 6)
│  │         → 检查主板短路(Step 12)
│  └─ NO ↓
│
├─ 开机10秒-1分钟关机？
│  ├─ YES → 释放静电 → 重插内存(单条测试)
│  │         → 拔硬盘试进BIOS → CMOS放电/恢复BIOS默认
│  │         → 检查电源适配器功率是否足够
│  └─ NO ↓
│
├─ 开机1-5分钟关机？
│  ├─ YES → 检查散热(风扇是否转) → 清灰换硅脂(Step 10)
│  │         → 进BIOS看CPU温度 → 检查电源稳定性
│  │         → 如果是渐进性的(越来越快关机) → 90%是散热问题
│  └─ NO ↓
│
├─ 能进BIOS但进不了系统？
│  ├─ YES → 硬盘可能故障 → 换硬盘/U盘启动PE测试
│  │         → 系统损坏 → 修复/重装系统
│  │         → 驱动问题 → 安全模式卸载最近安装的驱动
│  └─ NO ↓
│
├─ 进系统后几分钟关机？
│  ├─ YES → 安全模式测试(Step 16)
│  │         → 如安全模式稳定 → 驱动/软件问题
│  │         → 如安全模式也关 → 硬件问题(散热/电源/内存)
│  │         → 检查事件日志(Step 18)
│  │         → 杀毒(Step 22)
│  └─ NO ↓
│
└─ 不定时随机关机？
   → 电源适配器/插座电压不稳
   → 主板电容老化(目视检查)
   → 内存接触不良(重插)
   → 检查电源环境(接地线、老房子零火线问题)
```

## 第八章：机械革命无界14+ (7840HS) 型号专属问题与解法

> ⚡ **此章节针对你的具体机型**：机械革命无界14+，AMD Ryzen 7 7840HS，Radeon 780M核显

### 8.1 【高优先】BIOS版本过旧导致各种异常

**已知问题**：出厂BIOS存在多个Bug，会导致蓝屏、内存识别异常、随机重启等。

**解决方案**：
- 升级BIOS到 **T140_PHX_13** 或更新版本
- 升级方法：下载exe文件，双击运行即可升级（**升级过程中绝对不能断电/关机！**）
- BIOS下载：机械革命官网驱动下载页面，或百度网盘搜索"无界14+ BIOS"
- 2.26版本BIOS（网友分享）：百度网盘搜索关键词 `无界14+ 7840hs bios`
- 升级后可在BIOS中解锁：
  - **内存频率切换**（4800MHz ↔ 5600MHz）
  - **虚拟显存大小调整**（512MB → 2G/4G）

> ⚠️ 如果你的电脑能开机哪怕短暂时间，**优先升级BIOS**，这可能一次性解决所有问题！

### 8.2 【关键】AMD 780M核显驱动崩溃（掉驱动/黑屏/卡死）

**症状**：
- 使用中突然黑屏，几秒后恢复或直接重启
- AMD弹窗提示"驱动已停止响应并已恢复"
- 玩游戏/看视频时卡死
- 开机后进系统不久就黑屏关机

**根因**：AMD 780M的驱动兼容性问题是该平台已知通病

**解决方案（按优先级）**：
1. **使用OEM驱动，不用AMD官方最新驱动**
   - 去机械革命官网下载该机型专用驱动
   - 或使用Windows自动更新的驱动（反而更稳定）
2. **用DDU彻底卸载后重装**
   - 进安全模式 → 运行DDU(Display Driver Uninstaller) → 完全清除AMD驱动
   - 重启后安装OEM版本驱动
3. **更新BIOS**（见8.1）
4. **调整显卡电压**（高级操作，需MorePowerTool等工具）
5. **禁止Windows自动更新驱动**
   - gpedit.msc → 计算机配置 → 管理模板 → 系统 → 设备安装
   - 启用"防止安装与这些设备实例ID匹配的设备"
   - 添加AMD显卡的设备实例ID

### 8.3 内存频率导致蓝屏/不稳定

**背景**：
- 无界14+出厂默认将内存频率下调到 **4800MHz**（实际硬件支持5600MHz）
- 这是为了避免蓝屏做的保守设置
- 如果有人在BIOS中手动调高到5600MHz，可能导致不稳定

**排查**：
- 进BIOS检查当前内存频率设置
- 如果设置为5600MHz且不稳定 → **改回4800MHz**
- 如果一直是4800MHz → 排除此原因

### 8.4 随机重启伴随主板"嘎达"声

**症状**：
- 毫无征兆突然黑屏重启
- 重启瞬间能听到主板发出"嘎达"一声（继电器/电容声）
- 有时不重启也能听到嘎达声，但听到后离重启不远

**可能原因**：
- 主板供电回路问题（电容/MOS管）
- 电源适配器功率波动
- 内存兼容性问题

**解决尝试**：
- 更新BIOS
- 更新AMD芯片组驱动
- 检查/更换电源适配器
- 如持续出现 → **主板硬件故障，需送修**

### 8.5 电池异常（显示0%/255%/无法开机）

**症状**：
- 使用中电池突然显示0%或255%
- 风扇声音巨大
- 之后无法开机

**解决方案**：
1. 拔掉电源适配器，长按电源键30秒释放静电
2. 插上电源适配器（不开机），充电30分钟
3. 尝试开机
4. 如仍不行 → 拆开后盖，断开电池排线，仅用适配器供电开机
5. 如仍不行 → 电池管理IC故障，需送修

### 8.6 Modern Standby (S0) 睡眠问题

**症状**：
- 合盖后无法唤醒，只有电源灯闪烁
- 合盖一晚上耗电50%
- 唤醒后系统异常/关机

**解决方案**：
- BIOS中将睡眠模式从 **S0 (Modern Standby)** 改为 **S3 (Suspend to RAM)**
- 或通过注册表修改：
  ```
  HKLM\SYSTEM\CurrentControlSet\Control\Power
  PlatformAoAcOverride = 0 (DWORD)
  ```
- 重启生效
